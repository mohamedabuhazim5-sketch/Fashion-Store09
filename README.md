# Fashion Store Pro V25 Shopping Experience

نسخة V25 تضيف تجربة شراء أقوى وأقسام تساعد العميلة تختار وتكمل الطلب.

## التشغيل

```bash
npm install
npm run dev
```

## الروابط

- المتجر: `http://localhost:5173/`
- لوحة التحكم: `http://localhost:5173/#/admin`
- المفضلة: `http://localhost:5173/#/wishlist`
- تتبع الطلب: `http://localhost:5173/#/track`

## الجديد في V25

- Frequently Bought Together Section.
- Search Hints Section.
- Checkout Confidence Banner.
- Brand Values Cards.
- Premium Service Strip.
- تحسينات إضافية لكروت المنتجات والموبايل.
- الحفاظ على كل مميزات V24.
