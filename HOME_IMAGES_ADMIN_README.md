# Home Images From Admin

الجديد:
- تبويب جديد في لوحة التحكم باسم "صور الواجهة".
- تم حذف الاعتماد على صور Unsplash الافتراضية في أقسام الواجهة الأساسية.
- لو الصورة غير موجودة، يظهر مكان فارغ مكتوب عليه "أضيفي الصورة من لوحة التحكم".
- يمكن رفع الصور أو وضع رابط مباشر.
- مع Supabase: الصور تترفع على Storage ويتم حفظ الروابط في جدول site_settings.
- بدون Supabase: الصور تُحفظ في localStorage على نفس المتصفح فقط.

خطوات Supabase:
1. شغّل ملف SQL:
   fashion-store-home-images-settings.sql
2. افتح لوحة التحكم > صور الواجهة.
3. ارفع الصور أو ضع روابط الصور.
4. ستظهر الصور في الصفحة الرئيسية.
